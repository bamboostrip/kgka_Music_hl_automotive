# 安卓端KA-Music更新日志

## v2.4.0

- 应用名称更改为「时音」，统一全平台品牌标识
- 新增排行榜页面，支持浏览酷狗各类榜单及歌曲列表，车机横屏下自动切换为双列网格布局
- 新增播放状态持久化，重启应用后自动恢复上次播放的歌曲和进度
- 新增「我的」页面 VIP 徽章和到期时间展示，每日 VIP 领取成功后自动刷新
- 新增歌单多选模式，支持批量删除等操作
- 排行榜歌曲操作菜单新增「查看歌手」功能
- 车机模式左侧播放栏点击歌名/歌手名可直接进入播放详情页
- 优化排行榜页面排版，从网格卡片改为横向列表卡片设计，信息密度更高
- 优化 Rust 引擎内存占用，降低 IoT/车机设备内存压力
- 优化图片解码尺寸与应用暂停时缓存清理，减少后台内存占用
- 减少播放 UI 重建频率，优化播放队列内存开销
- 修复排行榜歌曲「...」按钮点击直接播放的问题，改为弹出操作菜单
- 修复排行榜播放后手机版底部 mini 播放卡片不显示的问题
- 修复搜索结果始终相同的问题（参数名不匹配）
- 修复 Windows 下中文文件名本地音乐无法播放的问题
- 修复歌单收藏类型、分页和删除相关的问题
- 修复点赞/取消点赞功能异常的问题
- 修复搜索页歌手头像无法显示的问题

## v2.3.0

- 新增车机模式（在设置-个性化中开关，开启后提供大字体与左侧常驻播放面板，支持自动检测 Android Automotive 车机设备）
- 新增可调节的播放缓存上限设置，可在设置-缓存管理中修改上限
- 新增连接蓝牙、有线耳机等新音频设备时自动恢复/开始播放的功能
- 新增基于 Android 原生 DynamicsProcessing 效果器的实时音量均衡功能，可在设置中开启以平衡歌曲音量差异
- 调整 App 更新行为，将原先在应用内下载并自动弹出安装修改为直接跳转至浏览器，由用户自主下载更新包
- 优化自定义背景图预缓存机制，彻底解决切换页面时背景闪白与 1s 左右的渲染延迟问题
- 修复车机模式下切歌歌词不刷新的问题，并移除方向盘长按暂停功能
- 修复歌词翻译悬浮开关与切换按钮无效的问题
- 优化歌词翻译对应关系，并修复和优化了其他已知问题

## v2.1.0

- 新增本地音乐检索与播放功能，支持自动检索并播放手机文件夹中的音乐和歌词
- 新增本地文件夹选择窗口，支持直接选择音乐目录，不用再手动输入路径
- 支持在未登录状态下设置 API URL 服务器地址，方便私有部署
- 新增宽屏专属侧边导航栏，大屏幕下自动由底部导航切换至侧边栏
- 优化大屏页面布局防止横向拉伸，首页、我的、设置等主要页面支持居中显示
- 优化大屏网格与双列布局，我的页面歌单支持网格显示，首页推荐歌曲支持双列并排
- 优化搜索页面，底部支持显示音乐播放控制栏
- 优化搜索历史记录即时同步，解决历史新词延迟一拍显示的问题
- 优化热搜榜单对齐与展示，修复热搜榜错位、被切掉与手势冲突的问题
- 修复定时播放中“播放完再停止”开启后再次进入设置回显为关闭的问题
- 修复平板/横屏下退出播放器详情页被强制锁定回竖屏的 Bug
- 修复本地无历史记录时进行搜索导致界面卡住、搜不出任何东西的问题
